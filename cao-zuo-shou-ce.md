# 操作手册

### AI识图界面说明 <a href="#ai-shi-tu-jie-mian-shuo-ming" id="ai-shi-tu-jie-mian-shuo-ming"></a>

产品界面主要包括两大部分。

第一部分是顶部导航栏，依次是：用量查看、消息通知、问卷反馈、帮助中心、个人中心。

![操作步骤](http://10.10.10.63:8449/help/matrialExtraction/ai-interface-top.png)

第二部分是操作区域，依次是：文件列表区、缩略图列表区、底图展示区、识别结果区。

![操作步骤](http://10.10.10.63:8449/help/matrialExtraction/ai-interface-main.png)

### [#](http://10.10.10.63:8449/help/AI/userguide/#%E6%93%8D%E4%BD%9C%E8%AF%B4%E6%98%8E)操作说明 <a href="#cao-zuo-shuo-ming" id="cao-zuo-shuo-ming"></a>

#### [#](http://10.10.10.63:8449/help/AI/userguide/#%E6%93%8D%E4%BD%9C%E6%B5%81%E7%A8%8B)操作流程 <a href="#cao-zuo-liu-cheng" id="cao-zuo-liu-cheng"></a>

AI识图·材料表产品操作流程较为简单，主要分为以下几步：

上传文件——自动识别——查看结果——核对修改——导出材料

#### [#](http://10.10.10.63:8449/help/AI/userguide/#%E4%B8%8A%E4%BC%A0%E6%96%87%E4%BB%B6)上传文件 <a href="#shang-chuan-wen-jian" id="shang-chuan-wen-jian"></a>

①点击【上传】-【单线图文件】

②从电脑中选择一个或多个单线图文件

③等待文件（依次）上传完成

![操作步骤](http://10.10.10.63:8449/help/matrialExtraction/ai-choose-file.png)

![操作步骤](http://10.10.10.63:8449/help/matrialExtraction/ai-file-list.png)

提示

打印版、正向的、有材料表表头的PDF单线图，平均识别准确率达95%；

若上传扫描版的单线图文件，可能因为清晰度不够、图纸歪斜等情况导致识别的准确率大幅降低。

#### [#](http://10.10.10.63:8449/help/AI/userguide/#%E8%87%AA%E5%8A%A8%E8%AF%86%E5%88%AB)自动识别 <a href="#zi-dong-shi-bie" id="zi-dong-shi-bie"></a>

①等待文件解析完成，即可看到图纸的缩略图

②已解析的文件按图纸的页码顺序自动进行识别

③在文件列表区可查看文件的识别进度，当同一个文件下的所有图纸均识别完成后，文件状态则从“识别中”转变为”识别完成“

![操作步骤](http://10.10.10.63:8449/help/matrialExtraction/ai-drawing-list.png)

![操作步骤](http://10.10.10.63:8449/help/matrialExtraction/ai-process.png)

#### [#](http://10.10.10.63:8449/help/AI/userguide/#%E6%9F%A5%E7%9C%8B%E7%BB%93%E6%9E%9C)查看结果 <a href="#cha-kan-jie-guo" id="cha-kan-jie-guo"></a>

①在缩略图列表选中已识别完成的图纸，即可看到对应的底图展示和识别结果。

②将光标定位在底图展示区，可通过鼠标滚轮或键盘触控板放大和缩小底图。

③一页图纸内材料信息较多时，可通过鼠标滚轮或拖动竖向滚动条调整表格展示区域。

④鼠标定位到识别结果的某个单元格，底图对应的区域会被自动选中且居中显示。

⑤点击左侧文件列表区域的【收起】按钮，可隐藏列表区，而放大右侧底图展示区和识别结果区的宽度。

⑥鼠标移入底图展示区和识别结果区之间的边界线，拖动出现的图标可进行上下区域的高度调整。

![](http://10.10.10.63:8449/help/matrialExtraction/ai-multitable.png)

[**#**](http://10.10.10.63:8449/help/AI/userguide/#%E5%90%88%E5%B9%B6%E8%A1%A8%E5%A4%B4)**合并表头**

![](http://10.10.10.63:8449/help/matrialExtraction/ai-multicells.png)

[**#**](http://10.10.10.63:8449/help/AI/userguide/#%E6%8E%A5%E7%BB%AD%E8%A1%A8)**接续表**

若某页图纸中材料表是上一页的接续，此时图纸中没有表头，但在识别结果展示时会被自动填上表头

![](http://10.10.10.63:8449/help/matrialExtraction/ai-jiexubiao.png)

#### [#](http://10.10.10.63:8449/help/AI/userguide/#%E6%A0%B8%E5%AF%B9%E4%BF%AE%E6%94%B9)核对修改 <a href="#he-dui-xiu-gai" id="he-dui-xiu-gai"></a>

AI提取的材料表信息无法保证100%的准确，因此需要人工进行识别结果的核对和修改。

可能出现的识别错误分为以下几类：

[**#**](http://10.10.10.63:8449/help/AI/userguide/#%E6%96%87%E6%9C%AC%E9%94%99%E8%AF%AF)**文本错误**

鼠标点击任一单元格，即可进入编辑状态，对照底图将错误的文本进行修改；点击识别结果区域的【确认】按钮，已修改的内容会进行保存。已核对过的图纸会出现相应的标识。

修改一张图纸的识别结果，组织内其他图纸中同一种材料的错误文本则会被自动批量修改，且核对和修改过的单元格会进行标绿显示；对于标绿的材料内容，只需核对序号和数量即可。

![文字错误](http://10.10.10.63:8449/help/matrialExtraction/ai-words-error.png)

![文字错误](http://10.10.10.63:8449/help/matrialExtraction/ai-words-error2.png)

[**#**](http://10.10.10.63:8449/help/AI/userguide/#%E8%A1%8C%E9%94%99%E8%AF%AF)**行错误**

a 若识别结果出现了少行的情况：

将鼠标移入行头色块上，会出现添加图标【+】，点击【+】即可在该行下方插入新的空行

![文字错误](http://10.10.10.63:8449/help/matrialExtraction/ai-rows-error.png)

b 若识别结果出现了多行的情况：

鼠标单击行头色块，会出现删除图标【×】，点击【×】即可将该行及其单元格内容清除，点击表格其他单元格可取消选中当前行；

鼠标从上而下拖动选中行头色块，点击【×】可批量删除选中的多行及其单元格内容，点击表格其他单元格可取消选中当前行。

![文字错误](http://10.10.10.63:8449/help/matrialExtraction/ai-delete-row.png)

![文字错误](http://10.10.10.63:8449/help/matrialExtraction/ai-delete-rows.png)

[**#**](http://10.10.10.63:8449/help/AI/userguide/#%E5%88%97%E9%94%99%E8%AF%AF)**列错误**

a 若识别结果出现了少列的情况：

将鼠标移入列头色块上，会出现添加图标【+】，点击【+】即可在该列右侧插入新的空列

![文字错误](http://10.10.10.63:8449/help/matrialExtraction/ai-add-column.png)

b 若识别结果出现了多列的情况：

鼠标单击行头色块，会出现删除图标【×】，点击【×】即可将该列及其单元格内容清除，点击表格其他单元格可取消选中当前列；

鼠标从左向右拖动选中列头色块，点击【×】可批量删除选中的多列及其单元格内容，点击表格其他单元格可取消选中当前列。

![文字错误](http://10.10.10.63:8449/help/matrialExtraction/ai-delete-column.png)

![文字错误](http://10.10.10.63:8449/help/matrialExtraction/ai-delete-columns.png)

提示

Crtl+Z可逐步撤销表格内的单元格编辑、行编辑、列编辑操作，Crtl+Y则可逐步恢复已撤销的内容。

[**#**](http://10.10.10.63:8449/help/AI/userguide/#%E8%A1%A8%E6%A0%BC%E9%94%99%E8%AF%AF)**表格错误**

若AI将其他非材料表的信息提取为一个独立的表格，点击表格左上角的小色块，会出现删除图标【×】，点击【×】即可将该表格及其单元格内容清除；点击表格其他单元格可取消选中当前表格。

![文字错误](http://10.10.10.63:8449/help/matrialExtraction/ai-delete-table.png)

注意

表格的删除是不可撤销和恢复的。

[**#**](http://10.10.10.63:8449/help/AI/userguide/#%E6%9D%90%E6%96%99%E8%A1%A8%E6%9C%AA%E6%88%90%E5%8A%9F%E8%AF%86%E5%88%AB)**材料表未成功识别**

若一个文件下前10页图纸的材料表均未成功识别，则需要手动框选出材料表区域，帮助AI重新进行识别。

step 1：材料表未成功识别会触发提示，点击【知道了】进行下一步操作

![](http://10.10.10.63:8449/help/matrialExtraction/ai-fail-step1.png)

step 2：点击【框选】图标，进入框选状态

![](http://10.10.10.63:8449/help/matrialExtraction/ai-fail-step2.png)

step 3：拖动鼠标出现适当大小的矩形来框选材料表区域，拖动四个顶点可调整矩形大小，点击删除图标【×】可取消对应矩形选框；若有多个材料表需分别进行框选

![](http://10.10.10.63:8449/help/matrialExtraction/ai-fail-step3.png)

step 4：点击【框选完成】按钮

![](http://10.10.10.63:8449/help/matrialExtraction/ai-fail-step4.png)

step 5：准确完整地框选出材料表区域后，确认操作，AI会重新进行识别

![](http://10.10.10.63:8449/help/matrialExtraction/ai-fail-step5.png)

提示

未识别出材料表的图纸在重新识别时，不占用额外的用量。

[**#**](http://10.10.10.63:8449/help/AI/userguide/#%E8%A1%A8%E5%A4%B4%E6%9C%AA%E6%88%90%E5%8A%9F%E8%AF%86%E5%88%AB)**表头未成功识别**

当图纸的表头未成功识别时，可能造成识别结果的行列错误，则需手动填写表头，帮助AI重新进行识别。

step 1：材料表表头未成功识别会触发提示，图纸本身有表头，选择【填写表头】，否则选择【忽略】

![](http://10.10.10.63:8449/help/matrialExtraction/ai-nobiaotou-step1.png)

step 2：点击【知道了】按钮，进入表头编辑状态

![](http://10.10.10.63:8449/help/matrialExtraction/ai-nobiaotou-step2.png)

step 3：将图纸上的材料表表头准确填写至识别结果表格中，列数错误时可【插入列】或【删除列】

![](http://10.10.10.63:8449/help/matrialExtraction/ai-nobiaotou-step3.png)

step 4：表头填写无误后，点击【填写完成】按钮

![](http://10.10.10.63:8449/help/matrialExtraction/ai-nobiaotou-step4.png)

step 5：确认操作，AI会重新进行识别

![](http://10.10.10.63:8449/help/matrialExtraction/ai-nobiaotou-step5.png)

#### [#](http://10.10.10.63:8449/help/AI/userguide/#%E5%AF%BC%E5%87%BA%E6%9D%90%E6%96%99)导出材料 <a href="#dao-chu-cai-liao" id="dao-chu-cai-liao"></a>

完成识别结果的核对修改后，即可将材料表导出。

①导出操作：

点击【导出】，则会将材料表以Excel格式的文件通过浏览器下载到电脑中；

若一个项目存在多个单线图文件，则导出时选择多个文件，可将材料表导出到同一份Excel中。

②材料表内容：

导出的Excel材料表包含两个sheet，【材料清单】是按顺序罗列每张图纸中的材料表信息，【材料汇总】是相同的材料信息的数量汇总。

![文字错误](http://10.10.10.63:8449/help/matrialExtraction/ai-export-button.png)

![文字错误](http://10.10.10.63:8449/help/matrialExtraction/ai-export-files.png)

![文字错误](http://10.10.10.63:8449/help/matrialExtraction/ai-material-list.png)

![文字错误](http://10.10.10.63:8449/help/matrialExtraction/ai-material-allin.png)

{% hint style="info" %}
文件识别完成之前不可导出材料表；批量导出不同项目的文件可能会导致材料表内容不准确。
{% endhint %}
